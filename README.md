### About Breakflow

Breakflow is a Chrome extension designed to enhance your experience while using Webflow's Designer. It provides productivity features to streamline your workflow and is a direct replacement for Finsweet's Webflow extension.

### Features include:
- Breakpoint hider: Hide unwanted breakpoints to declutter your workspace.
- Pixel to rem converter: Easily convert pixel values to rem units for better responsive design.
- Bonus copy CSS feature: Copy font scaling CSS to directly paste into your projects.

Have suggestions or feature requests? Feel free to open an issue on the GitHub repository.

### For faster extension updates, feel free to clone this repo and load into chrome unpacked.
